# Group Scholar Docs

Company-wide documentation site for Group Scholar. This repo hosts the public-facing static docs site and internal reference material.

## Structure

- `index.html`: Landing page
- `styles.css`: Site styles
- `docs/`: Documentation content (markdown)
- `ralph-progress.md`: Iteration log

## Local preview

Open `index.html` in a browser. Markdown docs can be read directly or rendered by a static-site pipeline in later iterations.
