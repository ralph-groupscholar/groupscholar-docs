# Operations Playbook

## Summary
This playbook documents how we run weekly operations, coordinate across teams, and keep work visible.

## Weekly rhythm
- Monday: priorities, risks, and milestones
- Wednesday: midpoint status check and unblockers
- Friday: outcomes review and next-week planning

## Communication norms
- Default to written updates in shared channels
- Summarize decisions in one paragraph
- Timebox synchronous meetings and capture actions

## Key artifacts
- Weekly priorities board
- Program launch checklist
- Incident log and postmortems

## Escalations
- Flag critical blockers within 24 hours
- Use the escalation template in the incident log
- Identify an owner, scope, and next update time
